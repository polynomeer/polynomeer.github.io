# Pure Function

컴퓨터 프로그래밍에서 순수함수(pure function)이란, 다음과 같은 속성을 갖는다.

1. 함수가 같은 어규먼트(입력)에 대해서 항상 같은 리턴값(결과)를 낸다.
2. 함수의 평가는 [사이드 이팩트](https://en.wikipedia.org/wiki/Side_effect_(computer_science))가 없다.
