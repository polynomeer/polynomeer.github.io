# 점 이력, 선분 이력

## 점 이력

점 이력은 특정 시점에서의 데이터 상태를 기록하는 방식이다. 데터의 변경 사항을 추적하고 해당 시점에서의 데이터 값을 유지한다.

## 선분 이력

선분 이력은 데이터의 변경 사항을 추적하면서 데이터의 유효 기간을 관리하는 방식이다. 각 데이터에는 시작 시간과 종료 시간이 포함된다.
